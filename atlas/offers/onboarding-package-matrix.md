# Onboarding Package Matrix

| Offer | Default mode | Optional mode | Onboarding emphasis |
|---|---|---|---|
| Atlas Knowledge | Assisted setup | Managed secure | source boundary confirmation and approved-source handling |
| Atlas Drafting | Self-serve | Assisted setup | tone, document family, and review rules |
| Atlas Meeting Flow | Self-serve | Assisted setup | output format and follow-up routine |

## Rule
Onboarding should collect only approved structured fields, not open-ended bespoke consulting briefs.

# Pricing and Setup Logic

## Pricing structure
Atlas should price as business software, not as generic consumer AI.

## Launch logic
- self-serve packages should be the default entry path
- assisted setup should be available where source boundaries or buyer confidence require it
- managed secure mode should be reserved for higher-trust cases and protected delivery requirements

## Commercial rule
The website should sell a defined workflow package first, then allow structured configuration within approved limits.

## Messaging rule
Price should be explained as a combination of:
- package value
- setup mode
- scope boundary
- optional guided support

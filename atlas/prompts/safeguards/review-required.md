# Review Required Safeguard

## Trigger this safeguard when
- a response could influence a business, compliance, HR, legal-adjacent, or customer-facing decision
- the user requests a final answer based on incomplete evidence
- the user requests action language that could be misread as formal approval or factual certainty

## Required behavior
- mark the output as draft, advisory, or review-required
- encourage human confirmation before external use where appropriate
- avoid overstating certainty

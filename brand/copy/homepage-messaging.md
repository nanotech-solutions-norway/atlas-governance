# Homepage Messaging Guidance

## Positioning line
Buy security-scoped AI work models for Norwegian teams.

## Messaging rules
- sell a specific job, not abstract AI capability
- explain scope and trust early
- avoid overclaiming autonomy or unsupported deployment depth
- make the next action clear: configure, preview, or request guided setup

## Proof priorities
- bounded workflows
- public-vs-secure clarity
- optional managed setup
- operational time savings

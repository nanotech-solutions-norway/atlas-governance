# UI Principles

## Design character
Atlas should feel professional, clear, trust-first, and operationally disciplined.

## Core principles
- one dominant idea per section
- generous whitespace
- strong heading hierarchy
- proof and trust modules near decision points
- restrained motion only where it improves orientation
- enterprise-light visual language, not flashy consumer AI aesthetics

## Color direction
- light canvas
- deep navy / slate base
- muted blue accent for primary actions
- restrained neutrals for support content

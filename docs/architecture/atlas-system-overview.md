# Atlas System Overview

## Purpose

Atlas is a Norway-first commercial system for selling bounded, security-scoped AI work models.

## Three-plane architecture

### 1. Wix commercial shell
- public website
- SEO pages
- pricing and package pages
- FAQ and proof content
- forms and onboarding entry points
- bounded preview/demo surfaces

### 2. GitHub governance plane
- package definitions
- configuration blocks
- prompt architecture
- policy documents
- Wix field dictionaries and route maps
- runtime payload contracts
- QA and release controls

### 3. Secure runtime plane
- authentication and entitlements
- webhook and API handling
- protected delivery
- audit logging
- tenant/package state
- future managed secure mode

## Launch principle

Atlas should launch publicly as a ChatGPT-first offer with clear public-vs-secure boundaries. The secure runtime is prepared structurally and activated for sensitive or managed cases.

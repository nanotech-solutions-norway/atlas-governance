# Rollback Checklist

- [ ] Identify the exact release or commit to revert
- [ ] Confirm whether the issue is public-shell only or touches runtime contracts
- [ ] Revert affected governance files in GitHub
- [ ] Restore last approved Wix mapping or content state
- [ ] Confirm runtime payload compatibility
- [ ] Log the rollback in CHANGELOG.md
- [ ] Open follow-up issue for root-cause correction

# Wix CMS Field Dictionary

## Purpose

This document defines the fields that may appear in the Wix CMS and associated public flows.

## Core fields

- `offerId` — internal package identifier
- `offerLabel` — public package name
- `roleId` — selected role block
- `workflowId` — selected workflow block
- `deploymentMode` — selected delivery mode
- `languageMode` — Norwegian-first / bilingual / English-first
- `sourceBoundary` — approved source scope
- `prohibitedDataClass` — excluded data class selection
- `ctaRoute` — next action route

## Rule

Do not introduce free-form governance-critical fields in Wix unless they are first approved and versioned in this repository.
